private fun setRecyclerViewLayoutManager(): Unit

RecyclerView 에 LayoutManager 를 등록해줍니다. LayoutManager 는 기본적으로 GridLayoutManager 를 사용하며 현재 칼럼의 갯수와 화면 방향, 비율 등을 고려하여 Dynamic 하게 변경됩니다.
